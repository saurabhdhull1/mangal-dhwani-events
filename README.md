# Mangal Dhwani Events — Demo Website

This is a single-page Next.js + Tailwind CSS demo website scaffold for "Mangal Dhwani Events" — a premium Indian shehnai player and live instrumental music event company. All images are dummy placeholders.

How to run (locally):

1. Install dependencies:

   npm install

2. Start dev server:

   npm run dev

This project uses Next.js and Tailwind CSS. The contact form uses local state only and does not send real data.
